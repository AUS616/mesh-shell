# Mesh-shell
A simple custom shell implemented in C with basic command support.

## Features

- Change directory (`cd`)
- List directory contents (`ls`)
- Exit the shell (`exit`)
- Display help information (`help`)
- Create directories (`mkdir`)
- Remove directories (`rmdir`)
- Clear the terminal screen (`clear`)
